# Portfolio Website

This is a professional, high-quality portfolio website designed for **AyoTunde OmoPariola**. It features a modern, dark-themed design with "Glassmorphism" elements, responsive layout, and interactive animations.

## Features
- **Hero Section:** Dynamic typing effect and introduction.
- **About Section:** Highlights skills and professional background.
- **Projects Section:** Showcases GitHub projects with links.
- **Contact Section:** Social links and email placeholder.
- **Responsive:** Works on desktop and mobile.
- **Custom Cursor:** Unique interactive cursor.

## How to Run

### Option 1: Open directly
Simply double-click `index.html` to open it in your web browser.

### Option 2: Using a local server (Recommended)
If you have Python installed:
```bash
python3 -m http.server
```
Then open `http://localhost:8000`.

If you have Node.js installed:
```bash
npx serve .
```

## Customization
- **Edit Content:** Modify `index.html` to update text, links, and projects.
- **Edit Styles:** Modify `style.css` to change colors, fonts, or layout.
- **Edit Logic:** Modify `script.js` to change animations or behavior.
